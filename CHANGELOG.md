## [1.0.0] - 2026-03-06

### Added
- AI-powered automation builder with Claude (Haiku 4.5)
- Visual dashboard for managing automations
- Gmail integration — read inbox, send emails, email-triggered automations
- Calendar integration — view events, create events, calendar triggers, free slot finder
- Browser automation — Playwright-powered Chromium with persistent login sessions
- MCP server for Claude Desktop and Claude Code integration
- Remote access via Telegram bot and WebUI dashboard
- WiFi-based automation triggers
- Scheduled automations with cron expressions
- Natural language schedule parsing ("every weekday at 9 AM")
- AppleScript engine for deep macOS integration
- Dark/light theme with system sync
- Cloudflare tunnel support for remote access
