# Getting Started

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/myoctopus-app/docs/releases/latest)
2. Open the DMG and drag **MyOctopus** to your Applications folder
3. Launch MyOctopus

## First Launch

On first launch, macOS will ask for permissions. MyOctopus needs:

| Permission | Why |
|-----------|-----|
| **Accessibility** | To control apps, type text, and arrange windows |
| **Automation** | To send commands to other applications |
| **Notifications** | To display notification actions |

Grant these in **System Settings > Privacy & Security**.

## Claude Setup

MyOctopus uses your existing Claude subscription for AI features.

1. Install [Claude Desktop](https://claude.ai/download) or [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
2. Log in with your Claude account
3. MyOctopus will automatically detect your session

Supported plans: Claude Free, Claude Pro, or Claude Max. Free works for basic use; Pro or Max recommended for heavy use.

## Gmail Setup (Optional)

To use email features (reading inbox, sending emails, email triggers):

1. Go to **Settings > Integrations > Gmail** in MyOctopus
2. Enter your Gmail address
3. Generate a [Google App Password](https://myaccount.google.com/apppasswords) (requires 2-Step Verification)
4. Enter the 16-character App Password
5. Click **Connect**

See [Gmail Integration](/integrations/gmail) for more details.
