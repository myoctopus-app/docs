## Features

**Automation Engine** — Build multi-step workflows using natural language. Open apps, run shell commands, control system settings, type text, and chain actions together.

**AI Chat** — Powered by Claude via the Agent SDK. Describe what you want in plain English and MyOctopus generates the automation for you.

**Gmail Integration** — Read your inbox, search emails, send replies, and create email-triggered automations. All via IMAP with App Passwords.

**Calendar Integration** — View upcoming events, create new ones, find free slots, and set calendar-based triggers.

**Browser Automation** — Control a Chromium browser with Playwright. Navigate pages, fill forms, click buttons, take screenshots, and read page content. Persistent login sessions (WhatsApp Web, social media, etc.).

**MCP Server** — Use MyOctopus automation tools directly from Claude Desktop or Claude Code. Install once, control your Mac from any Claude interface.

**Remote Access** — Control your Mac remotely via Telegram bot or a web dashboard. Secured with PIN authentication and Cloudflare tunnels.

**Smart Triggers** — Automate based on incoming emails, WiFi network changes, calendar events, or cron schedules.
