# Contributing to MyOctopus

Thank you for your interest in contributing to MyOctopus!

## Bug Reports

Found a bug? Please [open an issue](https://github.com/myoctopus-app/docs/issues/new?template=bug_report.md) with:

- MyOctopus version (from Settings)
- macOS version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Feature Requests

Have an idea? [Open a feature request](https://github.com/myoctopus-app/docs/issues/new?template=feature_request.md) with:

- What you want to accomplish
- How you currently work around it (if applicable)
- Any examples or mockups

## Security Vulnerabilities

Please report security issues privately. See [SECURITY.md](SECURITY.md).

## Code Contributions

MyOctopus is currently closed-source. If you're interested in contributing code, please reach out via [Discussions](https://github.com/myoctopus-app/docs/discussions).
